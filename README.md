# django_instagram
Simple instagram application on Django Framework

###### Run project  
```virtualenv -p python3 venv```  
```source venv/bin/activate```  
```cd instagram```  
```python3 manage.py migrate```  
```python3 manage.py createsuperuser```  
```python3 manage.py runserver```

###### Run tests  
```cd instagram```
```pytest -vvv```
